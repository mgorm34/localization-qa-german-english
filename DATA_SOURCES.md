# Data Sources and Licenses

All datasets used in this project are open and freely available.

## Primary Datasets

### 1. WMT DE–EN
- **Description**: Large-scale German–English parallel corpus
- **Source**: https://www.statmt.org/wmt24/
- **License**: Various open licenses per sub-corpus
- **Usage**: Primary training/evaluation data

### 2. OpenSubtitles DE–EN
- **Description**: Parallel subtitle corpus (German–English)
- **Source**: https://opus.nlpl.eu/OpenSubtitles.php
- **License**: Creative Commons (CC BY 4.0)
- **Usage**: Colloquial/informal language evaluation

## Optional Datasets

### 3. Universal Dependencies (UD) German Treebanks
- **Description**: Annotated German syntactic trees
- **Source**: https://universaldependencies.org/
- **License**: Creative Commons (CC BY-SA 4.0)
- **Usage**: Linguistic cues and grammar analysis

## Notes
- No proprietary or commercial datasets are used
- All data should be downloaded to the `data/` folder
- Do not commit raw data files to the repository
